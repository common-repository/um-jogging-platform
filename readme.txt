Social jogging platform which is an extension of the [Ultimate member](https://wordpress.org/plugins/ultimate-member) plugin. The users can track their progress and compete with each other for various badges. In addition, the users get access to a scientifically backed jogging advice.

== Installation ==
 
1. Install this plugin
2. Install ultimate member plugin
3. Activate the ultimate member plugin
4. Activate this plugin

== Description ==

Ultimate member social jogging platform is an extension of the [Ultimate member](https://wordpress.org/plugins/ultimate-member) plugin. The plugin has a range of features which are attractive to joggers and can increase the traffic on your website. The extended functionality of this plugin doesn't interfere with the current existing features of Ultimate member and are completely optional for your users. 

= Features =

* This plugin adds one additional tab to the user page and a leaderboards page
* The users can track their jogging progress with this plugin
* The leaderboard page stimulates competition and best performances are shown here
* The users are rewarded for their participation on the platform with points and various badges
* Progress can be easily visualized in graphs where data can be examined
* Users can get a personalized advice which is based on scientific fundaments
